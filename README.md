# project-management-knowledge

プロジェクト管理に関するナレッジ集。主に組込み・車載ソフト開発における実践的な知見をまとめる。

スキルマップの自己評価・集計を自動化する **スキル診断 Web アプリ** も同梱している（[`app/`](app/)）。

### 推進者が改善活動を進める標準的な順序

```
Step 1：現状把握
  └ 診断ガイド → 自プロジェクトが暗黙知状態かを確認
  └ スキルマップ → チームの現状スキルレベルをベースライン計測

Step 2：改善計画の設計
  └ 汎用改善計画ガイド → 5軸診断・仮説立案・Phase設計
  └ 期間別改善計画ガイド → 活動期間に応じた優先施策を絞り込む

Step 3：スポンサー合意（G0）
  └ 具体適用版 § 4-1-1 → 部長への報告資料の構成・Q&A
  └ 標準との対応マップ → A-SPICE等の権威づけとして活用

Step 4：チームへの展開
  └ チーム実践ガイド → ロール別のやること・RACI・最小ワークフローを共有
  └ 課題別解決策ガイド → 課題ごとの具体的な解決手順を参照

Step 5：継続・評価
  └ 汎用改善計画ガイド § 5 → 継続運用設計・メトリクス評価
  └ スキルマップ → 3ヶ月・6ヶ月後の再評価でギャップを確認
```

## スキル診断アプリ（`app/`）

スキルマップの手作業による入力・集計を自動化した Web アプリ。
SQLite（ローカル・認証なし）と Firebase Auth + Firestore（クラウド・Google 認証）を環境変数で切り替えられる。

### 起動方法

```bash
cd app && npm install   # 初回のみ
cd app && npm run dev   # http://localhost:3000
```

### 画面構成

| 画面 | URL | 内容 |
|------|-----|------|
| トップ | `/` | 名前・ロール入力で診断開始（新規 / 続きから） |
| スキル評価 | `/assessment` | 10カテゴリ・53項目を Lv0〜3 で評価 |
| 個人ダッシュボード | `/dashboard/[userId]` | レーダーチャート・ギャップ一覧・育成アクション・履歴比較 |
| チームダッシュボード | `/dashboard/team` | レーダーチャート・カテゴリ別平均・弱点サマリ・ヒートマップ |
| マスタ管理 | `/admin` | カテゴリ・スキル項目・ロール別目標 Lv の編集・ユーザー管理 |
| ユーザー設定 | `/settings` | 表示名・ロール変更・アカウント削除（Firestore モード） |
| ログイン | `/login` | Google サインイン（Firestore モード） |

詳細は [`app/CLAUDE.md`](app/CLAUDE.md) を参照。
