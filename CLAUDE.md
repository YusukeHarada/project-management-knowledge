# project-management-knowledge — CLAUDE.md

## リポジトリ概要

組込み・車載ソフト開発における暗黙知改善のナレッジ集と、スキルマップ自己評価を自動化する Web アプリを管理するリポジトリ。

- ルート直下：プロセス改善・スキル育成に関するマークダウンドキュメント群
- `app/`：スキル診断 Web アプリ（Next.js + SQLite）

---

## スキル診断アプリの起動

```bash
cd app && npm install   # 初回のみ
cd app && npm run dev   # http://localhost:3000
cd app && npm test      # ユニットテスト
cd app && npm run build # 本番ビルド確認
```

アプリの詳細な設計・実装ガイドは `app/CLAUDE.md` を参照。

## Claudeへの制約

- `app/data/skill.db` はバージョン管理しない（`.gitignore` 済み）
- スキルマップのマスタデータは `app/lib/domain/seed.ts` で管理。`スキルマップ.md` を変更した場合は `seed.ts` も合わせて更新する
- ドキュメントのリンクはリポジトリルートからの相対パスで記述する
